# سامانه مدیریت پرونده
یک سیستم مدیریت جامع که برای دیجیتالی کردن فرآیند ثبت سوابق مرکز مشاوره جوانه رشد توسعه داده شده است.
این برنامه به پزشکان این امکان را می‌دهد که مراجعین را ثبت کنند، شماره پرونده‌ها را به طور خودکار پیگیری کنند و سابقه بیمار را از طریق یک رابط کاربری ساده و پاسخگو مدیریت کنند.
